## How to host it for free

### Option A — Netlify (easiest)
1. Go to https://app.netlify.com/drop
2. Drag this entire folder onto the page
3. Netlify gives you a live link immediately (e.g. `yourname.netlify.app`)
4. Optional: in Site settings → Domain management, add your own domain name later if you buy one, or just keep the free netlify.app address

### Option B — Cloudflare Pages
1. Create a free account at https://pages.cloudflare.com
2. "Create a project" → "Upload assets" → upload this folder
3. You get a free `*.pages.dev` link

### Option C — GitHub Pages
1. Create a free GitHub account and a new repository
2. Upload `index.html` and the `images` folder to the repository
3. Go to Settings → Pages → set source to the main branch
4. Your site will be live at `https://yourusername.github.io/reponame`

Any of these three are completely free, with no ads, and support a custom domain later if you ever buy one (e.g. drjoorawon.mu).
